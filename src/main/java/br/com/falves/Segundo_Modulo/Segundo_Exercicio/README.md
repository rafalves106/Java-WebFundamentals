# O Catálogo de Preços (Arrays JSON e HTTP 200)

**Objetivo:** Aprender a retornar uma lista (array) de objetos em formato JSON.

**Cenário:** O front-end precisa listar os produtos em destaque na home do site.

**Tarefas:**

1. Crie uma classe `Produto` (id, nome, preco).
2. Crie uma `List<Produto>` com pelo menos 3 itens.
3. Monte o JSON manualmente representando um **Array**: `[{"id":1,...}, {"id":2,...}]`.
4. Envie a resposta garantindo que o navegador identifique como um arquivo JSON.
5. Verifique no navegador se os dados aparecem formatados (o Chrome/Edge costuma organizar JSONs automaticamente).
